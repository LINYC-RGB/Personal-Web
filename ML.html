🔹 機器學習基本觀念

🔹 機器學習（Machine Learning）是什麼？

機器學習是一種讓電腦透過資料學習，進而自行找出規律與模式，不需要人工明確編寫規則的技術。簡單說，就是「讓機器從經驗中學習」。

🔹 三種主要類型

類型 說明 舉例
監督式學習（Supervised Learning） 有「輸入資料」與「正確答案（標籤）」 圖片分類、房價預測
非監督式學習（Unsupervised Learning） 只有「輸入資料」，沒有答案 客群分類（Clustering）、主成分分析
強化學習（Reinforcement Learning） 機器透過「獎勵與懲罰」學習行為策略 玩遊戲、機器人走迷宮

🔹 常見名詞解釋

名詞 解釋
特徵（Feature） 資料的屬性或變數，例如房子的「坪數」、「樓層」、「地點」
標籤（Label） 正確答案，例如房價、是否有糖尿病
模型（Model） 機器學習的學習結果，可用來預測未知資料
訓練（Training） 將資料餵給模型進行學習的過程
測試（Testing） 使用沒見過的資料，評估模型的預測能力
過擬合（Overfitting） 模型記住了訓練資料細節，導致在新資料上表現不佳
欠擬合（Underfitting） 模型太簡單，無法學到資料的規律
損失函數（Loss Function） 測量模型預測結果與正確答案的差距
準確率（Accuracy） 正確預測的比例，常用於分類問題

🔹 常見演算法舉例

演算法 用途
線性回歸（Linear Regression） 預測數值，如房價、股價
邏輯回歸（Logistic Regression） 二分類問題，如判斷是否生病
決策樹（Decision Tree） 可視化分類與回歸
支持向量機（SVM） 二分類、找最佳分隔邊界
K最近鄰居（K-NN） 比較相似資料分類
K-平均（K-Means） 非監督式分群
神經網路（Neural Network） 模仿人腦學習模式，適用複雜資料

🔹 常見統計與評估指標解釋

📌 P值（p-value）

用途：檢定統計結果是否顯著。

意義：P 值小 → 結果不太可能是巧合，表示有統計顯著性。

常見閾值：p < 0.05 表示顯著。

舉例：若測試藥物對血壓的影響，p < 0.05 通常表示有效果。

📌 準確率（Accuracy）

公式：Accuracy = (TP + TN) / (TP + TN + FP + FN)

適用於類別分布均衡的情境。

📌 精確率（Precision）

公式：Precision = TP / (TP + FP)

關注錯殺代價高的情況（如垃圾郵件）。

📌 召回率（Recall）

公式：Recall = TP / (TP + FN)

關注漏判代價高的情況（如癌症篩檢）。

📌 F1 分數（F1 Score）

公式：F1 = 2 × (Precision × Recall) / (Precision + Recall)

當 Precision 和 Recall 都重要時使用，對不平衡資料集尤其有用。

📌 AUC-ROC

定義：ROC 曲線下的面積，範圍為 0.5 到 1.0。

用途：評估分類模型區分能力，尤其適合資料不平衡的情況。

🔹 混淆矩陣（Confusion Matrix）

實際為正例 實際為負例
預測為正例 TP（真正） FP（假正）
預測為負例 FN（假負） TN（真負）
