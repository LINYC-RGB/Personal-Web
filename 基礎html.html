基礎 HTML 語法教學

🔹 HTML 基本結構

<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>我的第一個網頁</title>
</head>
<body>
  <h1>歡迎來到我的網站</h1>
  <p>這是一段段落。</p>
</body>
</html>

這是 HTML5 文件的基本架構。

🔹 標題（<h1> ~ <h6>）

<h1>標題1</h1>
<h2>標題2</h2>

🔹 段落（<p>）

<p>這是一段文字內容。</p>

🔹 超連結（<a>）

<a href="https://example.com">前往 Example 網站</a>

🔹 圖片（<img>）

<img src="image.jpg" alt="示意圖片" width="300">

🔹 清單（列表）

無序清單：

<ul>
  <li>蘋果</li>
  <li>香蕉</li>
</ul>

有序清單：

<ol>
  <li>第一步</li>
  <li>第二步</li>
</ol>

🔹 表格（<table>）

<table border="1">
  <tr>
    <th>姓名</th>
    <th>年齡</th>
  </tr>
  <tr>
    <td>小明</td>
    <td>20</td>
  </tr>
</table>

🔹 表單（<form>）

<form action="/submit" method="post">
  <label for="name">姓名：</label>
  <input type="text" id="name" name="name"><br>
  <input type="submit" value="送出">
</form>

🔹 文字樣式

<b>粗體</b>
<i>斜體</i>
<u>底線</u>
<br>  <!-- 換行 -->
