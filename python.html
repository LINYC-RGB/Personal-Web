基礎 Python 運算

一、數學運算符（Arithmetic Operators）

運算符說明範例結果
+加法3 + 25
-減法5 - 32
*乘法4 * 28
/除法（浮點數）5 / 22.5
//整除5 // 22
%取餘數5 % 21
**次方2 ** 38

二、比較運算符（Comparison Operators）

運算符說明範例結果
==等於5 == 5True
!=不等於3 != 2True
>大於4 > 2True
<小於4 < 2False
>=大於等於5 >= 5True
<=小於等於3 <= 5True

三、邏輯運算符（Logical Operators）

運算符說明範例結果
and邏輯且True and FalseFalse
or邏輯或True or FalseTrue
not邏輯非not TrueFalse

範例：

age = 20
has_id = True

if age >= 18 and has_id:
    print("可以進入")
else:
    print("禁止進入")

四、常用邏輯結構

if 判斷式：

x = 10
if x > 0:
    print("正數")
elif x == 0:
    print("零")
else:
    print("負數")

for 迴圈：

for i in range(5):
    print(i)  # 輸出 0~4

while 迴圈：

count = 0
while count < 3:
    print("計數：", count)
    count += 1

三元運算子（條件運算式）：

x = 5
result = "正數" if x > 0 else "不是正數"
print(result)

五、運算優先順序（從高到低）

  1. () 括號
  2. ** 次方
  3. + -（正負號）
  4. * / // %
  5. + -（加減法）
  6. == != > < >= <=
  7. not
  8. and
  9. or
