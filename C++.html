📘 基礎 C++ 觀念整理

🔹 1. 程式基本結構

所有 C++ 程式都從 main() 函數開始，是程式的入口點：

#include <iostream>
using namespace std;

int main() {
    cout << "Hello, world!" << endl;
    return 0;
}

🔹 2. 變數與資料型別

使用不同型別可以儲存整數、字串、浮點數等資料：

型別用途範例
int整數int age = 20;
float單精度小數float pi = 3.14;
double雙精度小數double d = 3.14159;
char單一字元char c = 'A';
bool布林值（真假）bool isTrue = true;
string文字字串string name = "Tom";

綜合範例：

#include <iostream>
#include <string>
using namespace std;

int main() {
    int age = 18;
    string name = "Alice";
    cout << name << " is " << age << " years old." << endl;
    return 0;
}

🔹 3. 條件判斷（if / else）

根據條件執行不同區塊的程式碼：

int score = 85;
if (score >= 60) {
    cout << "及格" << endl;
} else {
    cout << "不及格" << endl;
}

🔹 4. 迴圈

while 迴圈：重複執行直到條件為假：

int i = 0;
while (i < 5) {
    cout << i << " ";
    i++;
}

for 迴圈：更簡潔的迴圈形式：

for (int i = 0; i < 5; i++) {
    cout << i << " ";
}

🔹 5. 陣列（Array）

用來儲存固定長度的相同型別資料：

int nums[3] = {10, 20, 30};
cout << nums[1]; // 輸出 20

🔹 6. 函式（Function）

封裝重複使用的邏輯，並可回傳值：

int add(int a, int b) {
    return a + b;
}

int main() {
    cout << add(3, 4); // 輸出 7
    return 0;
}

🔹 7. 類別（Class）與物件（Object）

C++ 是物件導向語言，可以建立類別來描述資料與行為：

class Student {
public:
    string name;
    int age;

    void sayHello() {
        cout << "Hi, I'm " << name << endl;
    }
};

int main() {
    Student s;
    s.name = "Amy";
    s.age = 19;
    s.sayHello();
    return 0;
}
